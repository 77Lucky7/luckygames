LuckyBrawl Remake - by LuckyLucky7*

Instructions: Use the arrow keys to move. Click to shoot. Hold to continue shooting. If you stop attacking, you'll be able to passively heal. Try to kill the enemy and get as many trophies as possible!

Note: This game is meant to be a remake of LuckyBrawl. I've created this game so that I can make a base and expand upon it by adding never-seen before attack mechanics. There's not much in it yet, but it's a work in progress.

Any comments? Post them on the LuckyGames official Discord server!
